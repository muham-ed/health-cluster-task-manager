# Health Cluster Task Manager

# Overview
A Full-Stack web application designed to manage tasks and directives for health clusters.  
Supports HQ and 20 health clusters with interactive workflow tracking.

# Features
 Create and manage tasks and directives
 Assign tasks between HQ and clusters
 Interactive workflow (Not Started → In Progress → Delivered → Reviewed → Approved)
 Full activity log with timestamps
 PostgreSQL database integration
 RESTful API built with Node.js and Express

# Tech Stack
 Node.js
 Express.js
 PostgreSQL
 JavaScript
 GitHub for version control

# Installation
1. Clone the repository
2. Run `npm install`
3. Configure `.env` file with database credentials
4. Run `npm start`
